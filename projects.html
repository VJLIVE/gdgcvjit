Smart Campus Navigator

Completed June 2023
View on GitHub

Project Description

The Smart Campus Navigator is an interactive mobile application designed to help students, faculty, and visitors navigate our university campus with ease. The app features augmented reality wayfinding, indoor navigation, and points of interest. Users can search for specific buildings, classrooms, or facilities and get turn-by-turn directions with AR overlays. The system also provides accessibility features for visually impaired users.

Technologies Used

Flutter Firebase ARCore Google Maps API Dart

Project Features

  • Augmented Reality navigation
  • Indoor positioning system
  • Accessibility mode for visually impaired
  • Event location integration
  • Offline map support
  • Building search with filters

Data Visualization Dashboard

In Progress Started: March 2023
View on GitHub

Project Description

The Data Visualization Dashboard is a powerful web application that transforms complex datasets into interactive, visually appealing charts and graphs. Designed for researchers and data analysts, it supports multiple data formats, real-time data streaming, and collaborative features. The dashboard allows users to create custom visualizations, apply filters, and share insights with team members. Our goal is to make data analysis accessible to non-technical users while providing advanced features for experts.

Technologies Used

React D3.js Node.js MongoDB TypeScript

Project Features

  • Interactive chart builder with drag-and-drop
  • Real-time data streaming support
  • Collaborative dashboard editing
  • Export visualizations in multiple formats
  • User authentication and permissions
  • Custom color schemes and themes
